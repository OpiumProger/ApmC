# AmpC
# Генерация ингибиторов AmpC β-лактамазы с помощью reinforcement learning и суррогатных ML-моделей

## Обзор проекта

Проект посвящен разработке генеративного пайплайна для создания новых потенциальных ингибиторов AmpC β-лактамазы - фермента, ответственного за устойчивость бактерий к β-лактамным антибиотикам. Комбинируя методы машинного обучения, reinforcement learning и молекулярного докинга, мы создаем систему, способную генерировать химически валидные и биологически активные молекулы с улучшенными свойствами.

---

## Цели проекта

1. Разработать пайплайн генерации новых химических структур с заданными свойствами
2. Обучить суррогатные ML-модели для быстрой оценки молекул
3. Использовать reinforcement learning для направленной генерации ингибиторов
4. Оценить качество сгенерированных молекул с помощью фильтрации и молекулярного докинга

---
